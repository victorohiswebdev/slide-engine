Project Framework

Aim & Project Objectives

🎯

Aim

To design and implement an integrated Edge AI framework capable of predicting irrigation needs and assessing plant health in real time using low-cost offline embedded hardware.

📌

Objectives

  • ✔ Develop a three-zone smart planter prototype
  • ✔ Build predictive irrigation intelligence using machine learning
  • ✔ Deploy vision AI for plant stress detection
  • ✔ Integrate sensors, automation, and dashboard monitoring
  • ✔ Reduce water waste and improve crop management
  • ✔ Demonstrate scalable low-cost smart farming architecture
